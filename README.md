# SmartTask - Full Stack Management System

אפליקציה מתקדמת לניהול ומעקב משימות, המשלבת טכנולוגיות Client-Side ו-Server-Side בסינכרון מלא.

## 1. אודות הפרויקט
מערכת זו מאפשרת למשתמשים לנהל את סדר יומם ביעילות. היא כוללת ממשק לניהול מחזור חיים מלא של משימה (CRUD), סיווג לפי קטגוריות ומעקב אחר סטטוסים בזמן אמת.

## 2. מבנה מסד הנתונים
המערכת מבוססת על שלוש ישויות מרכזיות:
* **מטלות (Tasks):** טבלת הליבה הכוללת כותרת, תיאור, תאריכי יצירה ויעד, ומפתחות זרים לסיווגים.
* **מצבי ביצוע (Statuses):** הגדרת שלבי המשימה (ממתינה, בביצוע, הושלמה).
* **סיווגים (Categories):** קטלוג המשימות לפי תחומי עניין (עבודה, אישי, דחוף).

## 3. לוגיקה ו-Stored Procedures
התקשורת מול ה-DB מתבצעת דרך פרוצדורות שמורות:
1. `sp_AddNewTask` - יצירת רשומה חדשה.
2. `sp_ModifyTask` - עדכון נתוני משימה קיימת.
3. `sp_RemoveTask` - מחיקה מהמערכת.
4. `sp_FetchAllTasks` - שליפה מרוכזת של הנתונים עם Joins.
5. `sp_LoadLookups` - טעינת רשימות הסטטוסים והסיווגים.

## 4. הוראות הרצה

### צד שרת (.NET Web API)
1. פתחי את הפתרון (Solution) ב-Visual Studio.
2. עדכני את ה-Connection String בקובץ `appsettings.json` לשרת המקומי שלך.
3. הריצי את ה-API (F5).

### צד לקוח (Angular)
1. פתחי טרמינל בתיקיית ה-Client.
2. הריצי `npm install` להתקנת הספריות.
3. הריצי `ng serve` והיכנסי לכתובת: http://localhost:4200.
